# useEventObservable

Returns an observable of events and a function to emit a new event.

function useEventObservable<TEvent>(): [events$: Observable<TEvent>, emit: Emit<TEvent>];

Shorthand Alias

This hook can be called with the shorthand alias useEvent$.
