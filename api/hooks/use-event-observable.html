# useEventObservable

Returns an observable of events (events$) and a function to emit a new event (emit).

function useEventObservable<TEvent>(): [events$: Observable<TEvent>, emit: Emit<TEvent>];

Like RxJS' Subjects (opens new window), whenever a new observer subscribes to the returned events$ observable, the observer listens for new events to be emitted. The instance of the observable will also persist across the components lifecycle to avoid issues with consumer hooks subscribing to a new instance on each render.

# Example

Example with useSubscription.

// Create an observable of click events and a function to emit a click event
const [click$, onClick] = useEventObservable();

// Listen for and log click events
useSubscription({
	next() {
		console.log('I was clicked');
	},
});

<button click={onClick} />;
